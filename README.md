### About Me

I am a passionate Android Developer focused on building scalable, maintainable, and high-performance mobile applications with modern architecture and clean UI.

- 💻 Specializing in native Android development using **Kotlin** and **Jetpack Compose**.
- 🏗️ Strong understanding of **OOPs** concepts and modern architectural patterns (MVVM, Clean Architecture).
- 🔌 Experienced with local persistence (**Room Database**, **SQLite**), backend integration (**REST API**, **Ktor**), and **Firebase**.
- 🛠️ Leveraging tools like **Hilt** for dependency injection and **Git** for version control.
- 🌱 Always eager to learn new technologies and build innovative solutions.
